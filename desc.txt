1. descripsi portfolio :
Ini adalah proyek personal branding yang saya buat dengan menggunakan Next JS, Tailwind CSS, dan Framer Motion. Proyek ini terdiri dari situs web yang menampilkan keterampilan, pencapaian, dan kepribadian saya. Situs web ini bertujuan untuk mengesankan dan terhubung dengan klien, pemberi kerja, dan kolaborator potensial.

contoh descripsi project:
1. HooBank adalah sebuah website yang saya buat dengan menggunakan React JS, Tailwind CSS, dan Vite. Website ini menampilkan informasi mengenai sebuah perusahaan yang bergerak di bidang digitalisasi keuangan, seperti layanan pembayaran, transfer, dan investasi online. Website ini juga memiliki fitur keamanan, kenyamanan, dan kemudahan yang memudahkan pengguna dalam mengelola keuangan secara digital.

2. OurTube adalah website yang saya buat menggunakan React JS dan Material UI. Situs web ini adalah tiruan dari YouTube untuk tujuan pembelajaran. Website ini memungkinkan pengguna untuk menonton dan mencari video dari berbagai kategori dan channel. Website ini juga memiliki fitur seperti rekomendasi. Website ini bertujuan untuk menunjukkan kemampuan saya dalam membuat aplikasi web yang kompleks dan fungsional dengan menggunakan teknologi modern.

3. Ini adalah tiruan Twitter yang saya buat dengan menggunakan Next JS, Typescript, dan Tailwind CSS untuk tujuan pembelajaran. Situs web ini memungkinkan pengguna untuk membuat, menyukai, dan me-retweet postingan dari pengguna lain. Situs web ini juga memiliki fitur seperti otentikasi. Website ini bertujuan untuk menunjukkan kemahiran saya dalam menggunakan Next JS, Typescript, dan Tailwind CSS untuk membuat aplikasi web yang dinamis dan responsif.

4. Elecar adalah situs web yang saya buat dengan menggunakan teknologi HTML, CSS, dan JavaScript. Situs web ini berisi tentang sebuah perusahaan yang menjual mobil listrik. Website ini memungkinkan pengguna untuk melihat-lihat, membandingkan, dan memesan berbagai model mobil listrik yang ramah lingkungan, efisien, dan bergaya. Situs web ini juga memiliki fitur seperti testimonial, FAQ, dan kontak. Situs web ini bertujuan untuk mempromosikan dan menjual mobil listrik perusahaan.
